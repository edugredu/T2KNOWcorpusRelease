# Source Selection Provenance

This note preserves the surviving Web of Science acquisition filters available for the T2KNOW project. These rows document acquisition-level provenance only. They are not a complete per-document source log and should not be interpreted as identifying each of the 821 reviewed T2KNOW-Core abstracts one-to-one.

The complete Web of Science export, database snapshot metadata, and document-level query assignment were not preserved in the current archive. The canonical release therefore starts from the reviewed annotations and provides internal document identifiers, split assignments, source-file names, BRAT paths, checksums, and corpus statistics for the released resource.

See `../source_selection_queries.tsv` for the preserved query/filter rows.
